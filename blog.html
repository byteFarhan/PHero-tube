The scope of var, let, and const!

var, let, and const are three keywords in JavaScript.
With these keywords we can declear variables in JavaScript. 'var' is only functional scoped keyword, which can create some issues in our program. 'let' and 'const' are block scoped keywords, This means they are accessible only within the block in which they are declared, whether it's a function, loop, or any other block. These keywords were introduced in the ES6 update in JavaScript.

The use cases of null and undefined!

null and undefined are two different data type or values in JavaScript.
null is used when we want to explicitly indicate the absence of a value to an empty state.
undefined is indicate the default value for variables that haven't been assigned a value. Function parameters that haven't been provided, or properties that don't exist in an object will give us undefined error. If we don't return any value from a function then function will return undefined as default value.

The meaning of REST API!

REST API is a way for computers to talk to each other over the internet.
It's like a waiter or menu at a restaurant, where we can ask for different dishes using simple rules. It helps apps and websites share information and do things together, like getting weather data or posting photos etc. REST is a set of rules that make this process organized and predictable. It helps different apps communicate and share data on the internet smoothly, just like ordering our favorite dish at a restaurant.
